# Portfolio Website - Ulin kT

A stunning one-page portfolio website built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## 🚀 Tech Stack

- **Next.js 14** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Framer Motion** - Smooth animations

## 📦 Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🎨 Features

- ✨ Smooth scroll animations with Framer Motion
- 🎯 One-page scrollable design
- 📱 Fully responsive layout
- 🌙 Dark theme with purple accents
- 🎭 Interactive laptop illustration with animations
- 🔗 Social media integration
- 💼 Portfolio showcase section
- 📧 Contact section

## 🛠️ Build for Production

```bash
npm run build
npm start
```

## 📝 Customization

Edit the content in `app/page.tsx` to personalize:
- Your name and title
- Social media links
- Portfolio projects
- Skills and technologies
- Contact information

Enjoy your new portfolio! 🎉
